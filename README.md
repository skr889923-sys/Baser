# دَلّني | Dallni - Campus Navigation System for the Visually Impaired

**دلّني (Dallni)** هو نظام ملاحي ذكي متكامل مصمم خصيصًا لمساعدة الطلاب المكفوفين وضعاف البصر على التنقل داخل الحرم الجامعي والوصول إلى مباني الكليات، القاعات، المكاتب، المصاعد، والمرافق الخدمية بأمان واستقلالية تامة.

يعتمد النظام على الملاحة التوجيهية الصوتية والاهتزازية خطوة بخطوة (Turn-by-Turn)، مع ربط التموضع الداخلي برموز الاستجابة السريعة (QR Codes) لضمان الدقة وتفادي انقطاع إشارة الـ GPS.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

### 📱 تطبيق الجوال (Mobile App)
- **React Native** & **Expo** (SDK 51)
- **Expo Router** (نظام التنقل القائم على الملفات)
- **TypeScript** & **Zustand** (إدارة الحالة الملاحية النشطة)
- **Expo Speech** (محرك التوجيه الصوتي) & **Expo Haptics** (أنماط الاهتزازات التوجيهية والتحذيرية)
- **Expo Location** (الموقع الجغرافي للمساحات الخارجية) & **Expo Camera** (مسح رموز QR للتموضع الداخلي)

### 💻 لوحة التحكم الإدارية (Web Admin Dashboard)
- **Next.js 14** (App Router)
- **TypeScript** & **Tailwind CSS**
- **Recharts** (الرسومات البيانية التفاعلية)
- **Lucide Icons** (الأيقونات البرمجية)

### 🗄️ قاعدة البيانات والخلفية (Database & Backend)
- **Supabase** (PostgreSQL Database)
- **Row Level Security (RLS)** (أمان البيانات وعزل الصلاحيات)
- **Database Migrations & Seed Data**

## 🚀 طريقة التشغيل والتهيئة المحلية

### ١. تنزيل التبعيات (Installation)
قم بتشغيل الأمر التالي في المجلد الرئيسي لتثبيت جميع التبعيات لكل الحزم والمشاريع تلقائيًا:
```bash
npm install
```

### ٢. تشغيل لوحة التحكم الإدارية (Admin Dashboard)
لتشغيل مشروع Next.js الإداري محليًا:
```bash
npm run dev:admin
```
ستعمل لوحة التحكم على الرابط المحلي: `http://localhost:3000`
*(يمكنك تسجيل الدخول ببريد ينتهي بـ `@ksu.edu.sa` وكلمة مرور مكونة من 6 خانات أو أكثر).*

### ٣. تشغيل تطبيق الجوال (Expo Mobile App)
لتشغيل تطبيق الجوال وتجربته عبر المحاكي أو المتصفح:
```bash
npm run dev:mobile
```
- اضغط `i` لتشغيل محاكي iOS.
- اضغط `a` لتشغيل محاكي Android.
- اضغط `w` لتشغيل نسخة الويب المخصصة للتجربة الفورية.

## 🕶️ مبادئ الوصول الشامل (Accessibility First)

تم بناء التطبيق الملاحة للجوال بالكامل بالتركيز على معايير الوصول الشامل:
1. **دعم قارئ الشاشة**: ربط جميع الأزرار بـ `accessibleLabel` و`accessibilityHint` باللغتين العربية والإنجليزية.
2. **التباين العالي (High Contrast)**: خيارات ألوان تباين أصفر وأسود وأحمر عالي الوضوح لضعاف البصر.
3. **التوجيه بالاهتزاز (Haptics)**: إرسال نبضات لمسية مميزة عند الانعطافات أو الاقتراب من الخطر.
4. **شاشات مسح واستعلام مبسطة**: أزرار تفاعلية ضخمة تملأ الشاشة لتسهيل التفاعل بدون النظر للجوال.

لمزيد من المعلومات، يرجى مراجعة المجلد المخصص للتوثيق في المجلد `/docs`.
